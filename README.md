# Sidh Cricket Academy - MERN Stack Booking System

A premium, full-stack cricket ground booking and academy management platform built with the **MERN stack** (MongoDB, Express, React, Node.js). Designed as a professional demo for high-performance sports facility management.

## 🚀 Professional Demo Features

- **Ground Booking**: Real-time availability calendar with morning, afternoon, evening, and night slots.
- **Confirmation Email Simulation**: Automatically shows a beautiful confirmation email modal after booking.
- **Admin Dashboard**: Comprehensive overview of revenue, booking stats, and customer data.
- **Academy Management**: Streamlined customer onboarding and player tracking.
- **Mobile Responsive**: Fully optimized for a premium experience on all devices.
- **Aesthetic UI**: Modern design with dark mode, smooth animations, and glassmorphism.

## 💻 Tech Stack (MERN)

- **M** - **MongoDB**: Flexible document storage (simulated for demo stability).
- **E** - **Express.js**: High-performance backend API.
- **R** - **React.js**: Dynamic frontend with Tailwind CSS & Framer Motion.
- **N** - **Node.js**: Robust server environment.

## 🛠️ Project Structure

```
├── frontend/               # React Application (Client)
│   ├── src/
│   │   ├── components/     # UI & Feature components
│   │   ├── pages/          # Application views (Home, Booking, Admin, etc.)
│   │   └── lib/            # Utilities & Mock data
├── backend/                # Express API (Server)
│   ├── storage.ts          # MERN Storage Layer
│   ├── routes.ts           # API Endpoints
│   └── index.ts            # Entry point
└── shared/                 # Shared Types & Schemas
```

## 🚀 Quick Start (Demo Mode)

1. **Install Dependencies**
   ```bash
   npm run install:all
   ```

2. **Run the Application**
   ```bash
   npm run dev
   ```

3. **Explore the Demo**
   - **Frontend**: `http://localhost:3000`
   - **Admin Dashboard**: `http://localhost:3000/admin`
